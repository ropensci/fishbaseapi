fishbase.ropensci.org

fishbase.ropensci.org is a RESTful Application Programming Interface (API) to the Fishbase database.

The Fishbase database and web portal are completely separate entities from this API. Fishbase is kind enough to give us a copy of their database, from which we provide some of it through this API.

Consider donating to Fishbase at fishbase.us/donate/

Source code for this API: https://github.com/ropensci/fishbaseapi



API documentation

find the docs at https://ropensci.github.io/fishbaseapidocs/
